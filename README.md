
  # AR Product Preview App

This project is a web-based Augmented Reality application that allows users to preview 3D products directly in their physical environment.

Using real-time camera input and spatial tracking, users can place and interact with virtual objects such as furniture or decor items, enabling a more informed and immersive shopping experience.

The application focuses on bridging the gap between digital and physical spaces by integrating WebXR and Three.js for smooth rendering and accurate object placement.

### ✨ Key Features
- 📷 Real-time camera-based AR experience  
- 📐 Accurate object scaling and placement  
- 🧱 Drag, rotate, and reposition 3D models  
- ⚡ Lightweight and optimized for mobile devices  
- 🌐 Runs directly in the browser (no app install required)  

  ## Running the code

  Run `npm i` to install the dependencies.

  Run `npm run dev` to start the development server.
